# Smart Queue System
## Deskripsi
Smart Queue System adalah aplikasi manajemen antrian digital yang memungkinkan pengguna mengambil nomor antrian dan memantau status layanan secara online untuk meningkatkan efisiensi dan kenyamanan dalam proses antrian.
## Tujuan
Tujuan dari sistem ini adalah membantu tempat layanan mengelola antrian secara digital sehingga proses pelayanan menjadi lebih efisien dan terorganisir.
## Target
- Klinik atau puskesmas
- Tempat layanan publik
- Usaha yang menggunakan sistem antrian
## Fitur
- Pengambilan nomor antrian secara online
- Melihat status antrian secara real-time
- Dashboard admin untuk memanggil antrian
- Riwayat antrian
## Teknologi
- HTML
- CSS
- JavaScript
- PHP / Node.js / atau framework lain
- Database MySQL
